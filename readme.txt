YouTube Plugin is the best YouTube Feed Plugin to Display clean, customizable, and responsive YouTube Channel Videos & YouTube Playlists.

== Description ==

YouTube plugin helps you to display YouTube Channel Videos feed from YouTube account, in same way you can also show YouTube Playlist feed into your posts, pages, home or in widget. With YouTube feed plugin, user can share your video Directly from your YouTube Channel Feed from your website. User can also comment on your YouTube Channel Videos directly form your website, same way user can like your YouTube Channel Videos from your website, no need to go to youtube.com website to comment or like your video. 

YouTube plugin pulls in your channel and videos from YouTube and shows them on your WordPress website. This is automatic. So when you add a new video on YouTube, you don’t have to update your site again

== Installation ==

The easy way :

* Go to the Plugins Menu in WordPress.
* Search for plugin "YouTube"
* Click "Install".
* After Installation click activate to start using the YouTube.
* Go to YouTube from Dashboard menu.

Not so easy way :

* To install YouTube via FTP
* Download the YouTube plugin
* Unarchive YouTube plugin
* Copy folder with add-youtube-feed.zip
* Open the ftp \wp-content\plugins\
* Paste the plug-ins folder in the folder
* Go to admin panel => open item "Plugins" => activate YouTube.
* Go to YouTube from Dashboard menu.
